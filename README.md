# Typing Speed Test in C

A console-based Typing Speed Test application developed in C. This program measures typing speed (Words Per Minute), typing accuracy, and total typing time.

## Features

- Select from 5 predefined sentences
- Measures Typing Speed (WPM)
- Calculates Characters Per Minute (CPM)
- Calculates Typing Accuracy
- Displays Total Typing Time
- Console-based Interface

## Technologies Used

- C Programming Language
- Standard C Library

## How to Compile

```bash
gcc Mainproject.c -o typing_test
```

## How to Run

### Linux/macOS

```bash
./typing_test
```

### Windows

```bash
typing_test.exe
```
